Color can be set to a particular element globally.

"class" selector is used to determine style by element class.

"id" selector is used to determine style by element id.

"^" symbol is used to select classes with their names starting with particular sequence of symbols.

"*" symbol is used to select all classes containing particular sequence of symbols.

This text is pink because the above applies.

Element style can be determined by a combination of consecutive attributes.

Style can be determined by element attribute.

It is possible to mark up a part of text only.

Styles can be applied to nested elements.

Style can be applied to

several elements at once.

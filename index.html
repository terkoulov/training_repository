Command list
  • git clone - clone folder from github.com
  • git status - view file status
  • git add . - add files to commit
  • git remote -v - view repository list
  • git commit -m "description" - commit new files + comment
  • git push origin branchname - push code to update a branch at github.com
  • git checkout -b branchname - create branch and switch thereto
  • git commit -am "description" - commit changes + comment
  • git log - view action log
  • git push origin HEAD - push files to update the head of current branch
  • git checkout branchname - switch to the specified branch
  • git pull origin branchname - pull code from a github.com branch
